What is Tesla Red Multi-Coat (PPMR)?

Red Multi-Coat is Tesla's premium deep-red paint, identified by the factory code PPMR. It is a multi-coat (tri-coat) finish, meaning the color is built from a reflective ground coat plus a translucent tinted coat, which gives it that rich, candy-like depth. Because it is a tri-coat, matching it well takes more than a single bottle of red — and that is exactly what our custom-mixed PPMR kit is built to do.
